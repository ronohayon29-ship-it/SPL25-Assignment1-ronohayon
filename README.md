# SPL Assignment 1 – DJ Session Management System

This project was developed as part of the **Systems Programming Laboratory (SPL)** course.

The system manages music playback sessions for DJs. It loads song data from input files and allows the user to interactively manage playlists and playback sessions.

The project focuses on **object-oriented design in C++**, dynamic memory management, and modular program structure.

---

## Project Structure

```
include/    Header files
src/        Source code implementation
input_2/    Example input files
Makefile    Build configuration
```

---

## Build

To compile the project run:

```bash
make
```

This will generate the executable in the `bin` directory.

---

## Run

After compilation run:

```bash
./bin/dj_manager
```

Alternatively you can run:

```bash
make test
```
